==============
arrayList.py 
==============
Creates an internal Array to store items of the capacity as well as funtions for pop, append, insert, shrink, grow, and the appropriate getters
and setters. When run, it tests all of the functionalities of the ArrayList class.

==============
arrays.py
==============
An Array is a restricted list whose clients can use
only [], len, iter, and str.

To instantiate, use

<variable> = array(<capacity>, <optional fill value>)

The fill value is None by default.
